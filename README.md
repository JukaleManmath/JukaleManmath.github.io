# Manmath Jukale — Portfolio

Personal portfolio website. Live at [JukaleManmath.github.io](https://JukaleManmath.github.io)

## Stack

Plain HTML5 + CSS3 + Vanilla JS. No frameworks, no build tools, no dependencies.

## Sections

- **Hero** — typewriter role ticker, GitHub contribution graph
- **About** — bio, stats, open-to roles
- **Education** — MS Computer Science (Binghamton), BE Computer Engineering (SPPU)
- **Experience** — 5 roles across healthcare, finance, and education
- **Projects** — 15+ projects across AI/ML, Backend, and Full Stack
- **Skills** — categorized skill chips with logos
- **Contact** — email, LinkedIn, GitHub with copy-to-clipboard

## Local Development

No setup needed. Just open `index.html` in a browser.
